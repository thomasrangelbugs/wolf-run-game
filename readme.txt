# Wolf Run

Wolf Run é um jogo web de corrida lateral desenvolvido para navegador, com controles responsivos para computador e celular. O jogador controla um lobo em uma jornada contínua, desviando de obstáculos, coletando moedas, usando pulo duplo e tentando alcançar a maior distância possível.

## Funcionalidades

* Jogo runner lateral diretamente no navegador
* Controle manual do personagem
* Pulo duplo
* Sistema de pontos
* Coleta de moedas
* Sistema de vida
* Registro de última pontuação e maior distância
* Mudança de clima
* Alternância de turno
* Sistema de estações
* Eventos especiais durante a progressão
* Sistema de estrelas
* Upgrades para melhorar o personagem
* Configurações de opacidade dos botões
* Ajuste de altura do jogo
* Opção de música ao iniciar
* Controles por teclado no computador
* Controles touch para celular
* Suporte a tela cheia
* Layout responsivo

## Objetivo

O objetivo do projeto é criar uma experiência de jogo simples, divertida e responsiva, explorando lógica de movimentação, colisão, pontuação, progressão, upgrades, eventos dinâmicos e adaptação para diferentes dispositivos.

## Tecnologias utilizadas

* HTML
* CSS
* JavaScript
* Netlify
